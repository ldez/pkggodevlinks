## Positions

- `readme.md`
- `<package>/readme.md`
- `godoc`
- `<package> godoc`

## Examples

```markdown
## Links

- [Title from readme 1](https://example.com)
- [Title from readme 2](https://example.org)
```

```go
// Package foo
//
// Links
//
// - Title from godoc 1, https://example.com
// - Title from godoc 2, https://example.org
package foo
```
